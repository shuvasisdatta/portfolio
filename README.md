## Tools used

```
 Vue.js v2 for frontend
```

```
 Vue Router for Routing
```

```
 Bulma as css framework
```

```
 fontawesome SVG icons for icons
```

```
 chart.js for chart
```

```
 particles.js for beautiful floating particles on homepage
```

```
 local database.json file for storage all data
```
